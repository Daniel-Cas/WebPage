Cronograma de actividades.

Diagrama de Gantt.

El Diagrama de Gantt es una herramienta gráfica la cuál usamos para visualizar el cronograma de actividades, donde el objetivo es exponer el tiempo de dedicación previsto para las diferentes tareas a desarrollar en el tiempo estipulado para el desarrollo del proyecto. Cabe resaltar que el Diagrama de Gantt no indica las relaciones existentes entre actividades.

Cronograma de actividades.

Las actividades planteadas para el adecuado avance con respecto al Proyecto se concretaron con un Diagrama de Gantt, el cual se realizó por Microsoft Project , el cual podemos fijar fechas establecidas de acuerdo con el avance que se necesite y la cronogloía con la que se realizarán, con respecto a la distribuición también nos ayuda organizando el equipo de trabajo y para tener en cuenta que realizaremos el Proyecto en el trascurso del IPA 2021.
