Proyecto Integrador

Este es un Proyecto Integrador el cual abarca diferentes C.A.D.I

Principalmente el Proyecto Integrador se inicia desde el Semestre V hasta el fin de corte que se tenga planeado en el calendario academico, por ende es de suma importancia la elaboracion del mismo por la agrupación, de los Campos de Aprendizaje Diciplinar para la demostración de lo aprendido en clases.

Los C.A.D.I que se trataran dentro de este proyecto de Software, el cúal cabe resaltar que será Web son: Contabilidad General - Ingenieria de Software I - Sistemas Operativos, obviamente las otras asignaturas se implementarán pero en un nivel más conceptual.

Teconologías Implementadas

Como ya sabemos el crear una Applicación Web desde cero es muy dificil; usamos las siguientes tecnologías diseñadas para el desarrollo rápido y eficiente.

Spring Boot

Tecnología Implementada de Java Maven, Un Framework Web para mejorar el desarrollo de Software.

Angular

Efectos, gestión de vistas y mejoras visuales con Angular en su verisón 11.

Mysql

Para la gestión de la Base de Datos Relacional, usaremos el gran motor MySql.
